Impact Theme Docs

About

A powerful theme for impactful websites

Beautiful and elegant company landing page theme this has a simple and minimalistic UI theme for hugo and is made with bulma and is Licensed under the MIT license

Getting Started

Add the theme as git submodule to your hugo project.

$ cd /path/to/hugo/project

$ git submodule add https://github.com/AcknologyHQ/impact-hugo.git themes/impact

Configure your theme in your hugo project.

example config: config.yaml

baseURL: "http://example.org"
languageCode: en-us
title: WEBSITE_TITLE

theme: impact

####
## Hugo Options
####

# Disable unwanted features
disableKinds:
    - RSS
    - taxonomy
    - term

disableHugoGeneratorInject: "true"

    minifyOutput: "true"
    tdewolff:
        html:
            keepComments: "true"
